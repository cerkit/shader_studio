# Shader Studio (MSL Animator)

Shader Studio is a tool for animating MSL (Metal Shading Language) shaders. It provides a built-in generative AI feature for creating shaders from a text prompt. It can then export a QuickTime movie of the animation. 


### Prerequisites

MSL Animator requires Swift 5.7 or later.  

### Getting Started
Download and install Google Antigravity IDE from https://antigravity.google

Clone the repository:

```sh
git clone https://github.com/cerkit/shader_studio.git
```

Open the project folder in Antigravity IDE.

Ensure you have the Swift language Plugin installed.

Open the terminal:

`CTRL` + `~`

```sh
cd shader_studio
```

Build the app:

```sh
swift build
```

Run the app:

```sh
swift run
```
